產生 public 和 private key


打開 git bash，到 /c/usr/your_name


cd /c/usr/your_name

建立 public 和 private key


ssh-keygen -t rsa -C "youremail@emailhost"

我們就可以在c:\users\your_name.ssh裡面找到public和private key

  • Private Key - id_rsa
  • Public Key - id_rsa.pub

將public key加入Github

進到 github profile，選擇左邊的 SSH and GPG keys，點選 New SSH key


將剛剛產生的 public key 填入並儲存 (位址應該是"C:\Users\yourname.ssh\idrsa.pub")
